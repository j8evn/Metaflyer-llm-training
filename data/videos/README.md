# 동영상 디렉토리

분석할 동영상 파일들을 이 디렉토리에 저장하세요.

## 지원 형식

- MP4 (.mp4)
- AVI (.avi)
- MOV (.mov)
- MKV (.mkv)
- WebM (.webm)

## 사용 방법

1. 동영상 파일을 이 디렉토리에 복사
2. 분석 실행:

```bash
python src/video_analyzer.py data/videos/your_video.mp4
```

## 권장 사항

- **해상도**: 720p 이상
- **길이**: 10분 이하 (테스트용)
- **형식**: MP4 권장
- **크기**: 1GB 이하 권장

## 분석 시간 예상

| 동영상 길이 | STT만 | STT + 멀티모달 |
|-------------|-------|----------------|
| 1분 | 30초 | 3-5분 |
| 5분 | 2-3분 | 15-20분 |
| 10분 | 5-6분 | 30-40분 |
| 30분 | 15-20분 | 2-3시간 |

## 최적화 팁

- 간격 늘리기: `--interval 5.0`
- 짧은 클립으로 테스트
- GPU 사용 권장
